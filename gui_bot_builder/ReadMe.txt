Before running this bot you need to run
Linux Users:
  sudo apt-get python3 - Get the software, if you don't already
  python3 -m pip install pyautogui - Needed to click and type
  sudo apt-get install python3-tk python3-dev - Needed to get mouse input
  python3 -m pip install numpy
  python3 -m pip install opencv-python - Needed to capture the screen and find relating images

Windows Users:
  python -m pip install pyautogui - Needed to click and type
  python -m pip install numpy
  python -m pip install opencv-python - Needed to capture the screen and find relating images
